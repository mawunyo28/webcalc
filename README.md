# [WebCalc](https://webcalc-omega.vercel.app/)

A simple website for a calculator.

## Motivation
I only just wanted to find out how simple I can make a website in the time constraint of two hours

## Tech Stack
- [React](https://react.dev/) -- UI library
- [Tailwindcss](https://tailwindcss.com/) -- utility-first styling
- [Vite](https://vitejs.dev) -- build tool and dev server

## Getting Started

### Prerequisites
- Node.js >= 18
- [pnpm](https://pnpm.io/) (Trust me: recommended)

### Installation
```bash
git clone https://github.com/mawunyo28/webcalc.git
cd webcalc
pnpm install 
```

### Developement
```bash
pnpm dev
```

open [http://localhost:5173](http://localhost:5173) in your browser.

### Build
```bash
pnpm build
```

### Preview Production Build
```bash
pnpm preview
```
